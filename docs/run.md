# Rodar localmente

Pré-requisitos: Node.js 18+ e npm/yarn instalados.

1. Instalar dependências na raiz (mono-repo):

   npm install

2. Rodar API:

   cd src/api && npm install && npm start

3. Rodar frontend:

   cd src/web && npm install && npm start

A documentação é gerada por MkDocs: `mkdocs serve` para desenvolvimento e `mkdocs build` para gerar site estático.
